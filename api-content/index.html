FreeSync（可变刷新率）：显示器的刷新率会随 GPU 输出帧率动态调整，不固定。 冲突： MPRT 的背光闪烁是基于固定帧时间设计的，占空比和闪烁周期通常与刷新率同步。 FreeSync 让每帧时长不固定，如果闪烁仍按固定周期，可能出现画面闪烁的情况。 BL_ADJ_PWM + VSync VSync（帧同步信号）：标记每帧刷新开始的时刻。 BL_ADJ_PWM：本来是亮度微调信号，让它根据 VSync 计数或定时调节占空比，就能实现： 背光闪烁严格与帧同步，即便刷新率变化也能正确对齐每帧。 动态调整 MPRT 的开关背光时间，保证 FreeSync 下每帧的运动模糊都可控且不闪烁。 简化逻辑： VSync 检测每帧起点。 背光根据 BL_ADJ_PWM 信号的占空比调节背光亮度，同时参考帧起点。 MPRT 背光闪烁周期随FreeSync刷新率变化动态调整。 11、SPI VSync
